Processor en Geheugen

RAM, ook wel intern geheugen of werkgeheugen genoemd, staat voor Random Access Memory. Deze kleine repen zijn een essentieel onderdeel van je computer, omdat je processor hier tijdelijke bestanden bewaart. Hoe meer RAM geheugen, hoe meer processen je tegelijkertijd uitvoert. Hoe sneller een processor is, hoe meer berekeningen en aan sturingen er per seconde kan verwerkt kan worden. Dus hoe meer taken de processor per seconde kan doen hoe sneller uw computer is.

RAM is een van de belangrijkste onderdelen binnen je computer. Zowel laptops, desktops, iMac, Macbook en NAS gebruiken RAM voor de tijdelijke opslag van programma's. Via het werkgeheugen voert je pc taken uit die door de processor verstuurd worden. Bijvoorbeeld wanneer je meerdere tabbladen open hebt, of 2 programma's naast elkaar draait. Daarom werkt een computer met weinig RAM geheugen langzamer dan een pc met veel intern geheugen.
